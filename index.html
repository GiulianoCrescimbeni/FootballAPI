Documentation

Parameters (those preceded by a ":") Description
championship The name of the championship that you want to search for; This are the example parameters: "seriea" "premierleague" "liga" "bundesliga" "ligue1"
squadname scorername The name of the squad/player you want to search for. You can also digit a bunch of letters, the API will fetch the corresponding results that contain them.
squadposition scorerposition This can be a single digit with the position of the player/squad that you are searching for in a specific league.
Squads Actions Commands Example
Get all the squads informations in a certain league /:championship/squads /premierleague/squads Result
Get a single squad by its name /:championship/squadname/:squadname /ligue1/squadname/brest Result
Get a single squad by its position /:championship/squadposition/:squadposition /seriea/squadposition/2 Result
Scorers Actions Commands Example
Get all the scorers informations from a certain league /:championship/scorers /liga/scorers Result
Get one or some scorers by searching their name /:championship/scorername/:scorername /bundesliga/scorername/lewandowski Result
Get a single scorer by its position /:championship/scorerposition/:position /seriea/scorerposition/5 Result
